# moq-web
Examples of bundler setups running [@moq/watch](https://www.npmjs.com/package/@moq/watch) and [@moq/publish](https://www.npmjs.com/package/@moq/publish).
All of the listed frameworks are setup using the Quick Start build with:

- NPM
- React
- Typescript
- Vue

If a framework is listed as supported, you can test it via:
```bash
npm i
npm run dev
```
